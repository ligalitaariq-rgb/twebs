# ZERO__BLOOD Website

## Overview
The ZERO__BLOOD website is an interactive platform designed to engage users and provide information about the ZERO__BLOOD community. This project includes a main HTML document, CSS for styling, and JavaScript for interactivity.

## Project Structure
```
zero-blood-website
├── index.html        # Main entry point of the website
├── css
│   └── styles.css    # Styles for the website
├── js
│   └── main.js       # JavaScript for interactivity
├── public
│   └── (static assets) # Directory for static assets like images and fonts
├── .gitignore        # Git ignore file
└── README.md         # Project documentation
```

## Setup Instructions
1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```
2. **Navigate to the project directory**:
   ```
   cd zero-blood-website
   ```
3. **Open `index.html` in your web browser** to view the website.

## Usage
- Enter a player's account name in the input field and click the button to verify membership in the ZERO__BLOOD community.
- The website displays the corresponding details or a message indicating if the name is not a member.
